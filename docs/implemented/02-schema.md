## Client

```prisma
model Client {
  clientId     String   @unique      // "clt_" + 24 hex chars, generated by Sentinel
  clientSecret String?               // bcrypt hash. null for public clients (SPAs)
  name         String                // human label, e.g. "My React App"
  isPublic     Boolean  @default(true)
  redirectUris String[]              // exact URIs allowed ‚Äî validated on every /authorize
  scopes       String   @default("openid")
  tokenFormat  String   @default("jwt")  // "jwt" | "opaque"
}
```

**Created by:** `POST /admin/clients` ‚Üí `services/clientService.js:registerClient()`

**Used in:**
- `controllers/oauthController.js:authorize` ‚Äî validates `client_id` and `redirect_uri`
- `services/authService.js:exchangeCodeForToken` ‚Äî reads `tokenFormat` to decide JWT vs opaque
- `services/authService.js:refreshAccessToken` ‚Äî reads `tokenFormat` for the new access token

## AuthCode

```prisma
model AuthCode {
  code                String   @unique   // crypto.randomBytes(32).toString('hex')
  userId              Int                // FK ‚Üí User
  clientId            String             // FK ‚Üí Client
  redirectUri         String             // must match what's in Client.redirectUris
  codeChallenge       String             // BASE64URL(SHA256(code_verifier)) ‚Äî stored here
  codeChallengeMethod String  @default("S256")
  scope               String  @default("openid")
  expiresAt           DateTime           // now + 5 minutes
  used                Boolean @default(false)  // one-time use, set true after exchange
}
```

**Created by:** `services/authService.js:createAuthCode()` ‚Äî after successful login

**Consumed by:** `services/authService.js:exchangeCodeForToken()` ‚Äî PKCE verified here, then `used = true`

## Token

```prisma
model Token {
  token     String   @unique   // JWT string (eyJ...) or opaque hex
  userId    Int                // FK ‚Üí User
  clientId  String             // FK ‚Üí Client
  scope     String
  expiresAt DateTime           // 15 min for JWT, was 24h for opaque
}
```

Stores every access token issued. JWT tokens are stored as their full string.

**Note:** For JWT clients, the `Token` row exists but validation doesn't use it ‚Äî JWT is verified locally via JWKS. The row is there for audit / future revocation list use.

## RefreshToken

```prisma
model RefreshToken {
  tokenHash String   @unique   // SHA-256(raw token) ‚Äî raw never stored
  userId    Int
  clientId  String
  scope     String
  expiresAt DateTime           // 30 days
  used      Boolean @default(false)  // rotation: true after first use
}
```

**Why hash?** If the DB leaks, raw refresh tokens can't be used. SHA-256 is enough here ‚Äî refresh tokens are already 40 random bytes (high entropy), no need for bcrypt's slowness.

**Rotation:** On use, `used = true`. New token issued. If the same token is used twice, it means someone else used it first ‚Äî detectable as potential theft.

**Created by:** `services/authService.js:issueTokenPair()`

**Consumed by:** `services/authService.js:refreshAccessToken()`

**Deleted by:** `services/authService.js:revokeRefreshToken()` (logout)
